Vidya Set

Vidya Setu is an educational web platform developed to connect students with learning resources and digital guidance in one place. The platform provides an easy and user-friendly environment where users can register, log in, access study materials, courses, and track their learning progress.

The project uses a Flask backend API to manage and store user data securely in the database. It handles important operations such as user registration, login authentication, profile management, and data storage. The backend ensures smooth communication between the frontend and database through RESTful APIs.

The frontend is designed with modern web technologies to provide a responsive and interactive experience for users. The system is built to make education more accessible, organized, and efficient.
